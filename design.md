# تصميم تطبيق موصل رايد (Mosul Ride)

## الهوية البصرية

**الاسم:** موصل رايد (Mosul Ride)
**الشعار:** سيارة مع خطوط حركة تعبر عن السرعة والموثوقية
**الألوان الرئيسية:**
- الأزرق الداكن: `#1A2E4A` (الثقة والأمان)
- الذهبي: `#F5A623` (الحيوية والطاقة)
- الأبيض: `#FFFFFF` (النظافة والوضوح)
- الرمادي الفاتح: `#F5F7FA` (الخلفيات)

## قائمة الشاشات

### شاشات المستخدم (Passenger App)
1. **Splash Screen** — شاشة البداية مع الشعار
2. **Onboarding** — 3 شرائح تعريفية بالخدمات
3. **Login / Register** — تسجيل الدخول برقم الهاتف + OTP
4. **Home Map** — الشاشة الرئيسية مع الخريطة وزر الحجز
5. **Book Ride** — اختيار نوع الخدمة (تكسي / توصيل / اشتراك) وتحديد الوجهة
6. **Driver Matching** — شاشة انتظار السائق مع التتبع الحي
7. **Active Ride** — تتبع الرحلة الحية مع معلومات السائق
8. **Trip Summary** — ملخص الرحلة والتقييم والدفع
9. **Delivery** — طلب توصيل طرد مع تفاصيل المرسل والمستلم
10. **Subscriptions** — خطط الاشتراك الشهري (موظفون / طلاب)
11. **Profile** — الملف الشخصي وإدارة الحساب
12. **Ride History** — سجل الرحلات السابقة
13. **Wallet** — المحفظة الرقمية والمدفوعات
14. **Support** — الدعم والمساعدة

### شاشات السائق (Driver App)
1. **Driver Onboarding** — تسجيل السائق مع رفع الوثائق
2. **Driver Home** — الشاشة الرئيسية للسائق (Online/Offline)
3. **Incoming Request** — طلب رحلة جديد مع قبول/رفض
4. **Active Trip** — تفاصيل الرحلة الجارية والملاحة
5. **Driver Earnings** — إحصائيات الأرباح اليومية/الأسبوعية
6. **Driver Profile** — الملف الشخصي والوثائق

## تدفقات المستخدم الرئيسية

**تدفق حجز التكسي:**
Home Map → اضغط "احجز رحلة" → اختر التكسي → حدد الوجهة → أكد الحجز → انتظر السائق → تتبع الرحلة → وصول وتقييم

**تدفق طلب التوصيل:**
Home Map → اضغط "توصيل طرد" → أدخل تفاصيل الطرد → حدد العنوان → أكد الطلب → تتبع السائق

**تدفق الاشتراك:**
Home → الاشتراكات → اختر الخطة → أدخل مسارات التنقل اليومي → ادفع → نشط

## الاختيارات التصميمية

- **الاتجاه:** RTL (عربي) بالكامل
- **الخط:** Cairo (عربي) + Inter (إنجليزي)
- **الأيقونات:** Material Icons + أيقونات مخصصة
- **الخريطة:** Google Maps / OpenStreetMap
- **الأنيميشن:** Reanimated 4 للانتقالات السلسة
- **الكروت:** زوايا مدورة 16px، ظل خفيف
- **الأزرار:** مستديرة كاملة للأزرار الرئيسية، مستطيلة مدورة للثانوية
