# إعداد Expo Project ID - خطوة خطوة

## الخطوة 1️⃣: افتح Settings في Expo Go

**على جهازك (الهاتف):**

1. انظر إلى الشاشة الحالية (Expo Go)
2. في الأسفل، ستجد 3 أزرار:
   - Home (البيت 🏠)
   - Diagnostics (الموجة 📡)
   - Settings (الترس ⚙️)
3. **اضغط على Settings (الترس ⚙️)**

**بعد ما تضغط:**
- ستظهر صفحة جديدة فيها معلومات حسابك
- ابحث عن **Username** أو **Account**
- **انسخ اسم المستخدم** (مثلاً: `mohammed_ali` أو `user123`)

---

## الخطوة 2️⃣: افتح متصفح على جهازك

**على نفس الجهاز (الهاتف):**

1. افتح أي متصفح (Chrome, Safari, Firefox, إلخ)
2. في شريط البحث، اكتب: `expo.dev`
3. اضغط Enter

**ستظهر صفحة Expo الرئيسية**

---

## الخطوة 3️⃣: سجل دخول إلى expo.dev

**على صفحة expo.dev:**

1. ابحث عن زر **Sign In** أو **Log In** (عادة في الأعلى اليمين)
2. **اضغط عليه**
3. ستظهر صفحة تسجيل الدخول

**في صفحة التسجيل:**
1. اكتب **نفس اسم المستخدم** الذي نسخته من Settings
   - أو البريد الإلكتروني الذي استخدمته في Expo Go
2. اكتب **كلمة المرور**
3. اضغط **Sign In**

**بعد تسجيل الدخول:**
- ستظهر صفحة Dashboard أو Projects
- هذا يعني أنك سجلت دخول بنجاح ✅

---

## الخطوة 4️⃣: أنشئ مشروع جديد

**على صفحة Dashboard:**

1. ابحث عن زر **Create a new project** أو **New project** أو **+**
   - عادة يكون في الأعلى أو في الوسط
2. **اضغط عليه**

**ستظهر نافذة إنشاء مشروع:**

1. في حقل **Project Name**، اكتب: `mosul-ride-app`
2. في حقل **Workflow**، اختر: **Managed Workflow**
3. اضغط **Create** أو **Create Project**

**بعد الإنشاء:**
- سيستغرق بضع ثوان
- ستظهر صفحة المشروع الجديد ✅

---

## الخطوة 5️⃣: انسخ Project ID

**على صفحة المشروع:**

1. ابحث عن **Project ID** (عادة يكون في الأعلى أو في قسم **Settings**)
2. **انسخ الـ ID بالكامل**
   - يبدو هكذا: `a1b2c3d4-e5f6-7890-abcd-ef1234567890`
   - أو: `12345678-abcd-efgh-ijkl-mnopqrstuvwx`

**تأكد من:**
- ✅ نسخت الـ ID بالكامل (بدون مسافات)
- ✅ الـ ID يحتوي على 36 حرف (أرقام وحروف وشرطات)

---

## الخطوة 6️⃣: أرسل لي الـ Project ID

**الآن:**
1. انسخ الـ Project ID
2. أرسل لي الـ ID في الشات
3. أنا سأضيفه في ملف `app.config.ts`
4. الإشعارات تعمل فوراً ✅

---

- استخدام **نفس الحساب** في Expo Go و expo.dev
- اسم المشروع يكون بالضبط: `mosul-ride-app` (بدون مسافات)
